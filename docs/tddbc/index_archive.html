TDDBC

ようこそ

TDD Boot Camp(TDDBC) とは、テスト駆動開発(Test Driven Development)について、座学だけでなく、実習形式で手を動かして体得することを目的とするイベントです。

各地のコミュニティの方々が中心となって、全国各地で行われています。

これから開催されるTDD Boot Camp

開催情報については、https://tddbc.connpass.com/ をご確認ください。
