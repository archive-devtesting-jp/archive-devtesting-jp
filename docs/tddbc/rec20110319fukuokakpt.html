TDDBC福岡 SCM企画KPT

SCM企画のKPT作成にご協力ください

TDDBC福岡では新企画として２日目にSCM学習を行いました。

この新企画について開催者、講師、参加者、様々に感じたことがあるかと思われます。

そこで、SCM企画について皆様の感じたことをKPTとしてここに集めたいと考えました。

TDDBCの更なる発展のため、ご協力いただければ幸いです。

Keep

  • SCM企画は続ける
  • DVCSの利用
  • 「最低限必要なコマンド」の説明
  • bleisさんの運用環境を聞けたのはよかった

Problem

  • デモがほしい
  • 環境構築に手間取ったペアがあった点
  • 説明の再確認を容易にする
  • ペアプロはSCMの特色が薄かった気がした
  • TDD+SCMならではの利点が見え辛かった

Try

  • 他SCMの導入
  • 1日目の実習から取り入れてみる
  • BTS/ITSとの連携
