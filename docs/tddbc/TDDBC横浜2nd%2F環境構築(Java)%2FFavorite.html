TDDBC横浜2nd/環境構築(Java)/Favorite

Eclipseのコンテントアシストの設定を行うと、JUnit4のアサーションメソッドやhamcrestのMatcherを使用する際にstaticインポートの入力が省略できるため、コーディングを快適に行うことができます。

1.EclipseのPrefrernceの画面で、「Java→Editor→Conetnt Assist→Favorite」を選択し、続いて「New Type」を選択します。

2ポップアップで、「Browse」を選択します。

3.「Type Selection」のポップアップで、「org.junit.Assert」を選択し、「OK」を選択します。

4.「OK」を選択します。

5. 2～4を繰り返し、3で「org.hamcrest.CoreMatchers?」と設定します。

6. スクリーンショットのようになっているのを確認し、「OK」を選択します。
