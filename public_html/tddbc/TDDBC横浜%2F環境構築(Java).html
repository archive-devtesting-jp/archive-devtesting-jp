TDDBC横浜/環境構築(Java)

JavaSDKの設定

JavaSDKのバージョンを確認してください。

$ javac -version
javac 1.6.0_24

バージョンが1.6.0未満である場合は、最新のJDKをインストールしてください。

Eclipseのインストール

http://www.eclipse.org/ からEclipse（Indigo）をダウンロードします。 パッケージは「Eclipse IDE for Java Developers」で構いません。 ダウンロード後は適当なディレクトリに展開してください。

Eclipseプラグインの設定

QuickJUnit? をインストールします。 「Help - Eclipse Marketplace」から検索し、「Install」ボタンを押してインストールできます。
