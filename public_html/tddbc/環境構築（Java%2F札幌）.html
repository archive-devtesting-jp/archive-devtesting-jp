環境構築（Java/札幌）

JavaSDKの設定

JavaSDKのバージョンを確認してください。

$ javac -version
javac 1.6.0_24

バージョンが1.6.0以下である場合は、最新のJDKをインストールしてください。

Eclipseのインストール

http://www.eclipse.org/ からEclipse（Helios）をダウンロードします。 パッケージは「Eclipse IDE for Java Developers」で構いません。 ダウンロード後は適当なディレクトリに展開してください。

Eclipseプラグインの設定

QuickJUnit? と MercurialEclipseをインストールします。 それぞれ、「Help - Eclipse Marketplace」から検索し、「Install」ボタンを押してインストールできます。

Bitbucketのアカウント作成

Bitbucket（https://bitbucket.org/ ）のアカウントを持っていない人は作成してください。 BitbucketはMercurialを用いたプロジェクトホスティングサービスで、プロジェクトの共有に使用します。
